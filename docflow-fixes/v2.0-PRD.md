## 1. Vize

DocFlow 2.0 je **nativní Android aplikace** pro správu smluv, předplatného a opakujících se výdajů. Postavena na **Capacitoru** (web kód v native shell), s plnou integrací Android 15 funkcí, biometrickým zámkem a scanováním smluv kamerou. Filozofie: **dostupné, bez reklam, jednou zaplaceno**.

## 2. Cíle DocFlow 2.0

| Cíl | Měřitelný úspěch |
|-----|-------------------|
| **Profesionální native zážitek** | Native splash, themed icon, Material You |
| **Bezpečnost** | Face ID/otisk pro odemčení, auto-lock |
| **Produktivita** | Scan smlouvy kamerou (OCR auto-fill) |
| **Notifikace** | FCM push s vibrací, action buttons |
| **Monetizace** | Jednorázových 15 Kč, žádné reklamy, žádné IAP |
| **Android 15** | targetSdk 35+, edge-to-edge, predictive back |

## 3. Funkce — co zůstane, co přibude

### ✅ Zůstává z TWA verze
- Login (Google, Apple, e-mail)
- Dashboard (přehled výdajů měsíční/roční)
- Seznam smluv s filtry, řazením, vyhledáváním
- 8 jazyků (CS, EN, DE, FR, ES, PL, ZH, HI)
- Dark / Light / System theme
- CSV export
- Family sharing (do 5 členů)
- Service Worker (offline fallback)
- Smazání účtu

### 🆕 Nové native funkce (DocFlow 2.0)

#### 🔐 Biometric Lock
- Face ID / otisk prstu pro odemčení apky
- Auto-lock po 5 / 15 / 60 minutách (volba)
- Hide screen v recent apps
- **Plugin:** `@aparajita/capacitor-biometric-auth`

#### 📸 Scan smluv kamerou
- Klik „+" → výběr „Scan smlouvy"
- Kamera otevře document scanner s auto-detekcí okrajů
- OCR rozpozná: název, datum, částku, frekvenci
- Předvyplní formulář, uživatel jen potvrdí
- **Plugin:** `@capacitor-mlkit/document-scanner` + `@capacitor-mlkit/text-recognition`

#### 🔔 Push notifikace
- Server-side push přes Firebase Cloud Messaging
- Ranní push (9:00) o smlouvách obnovujících se za 7 dní
- Vibrace (vlastní vzor)
- Action buttons v notifikaci („Zaplaceno" / „Posunout")
- **Plugin:** `@capacitor/push-notifications` + `@capacitor/haptics`

#### 🎨 Material You themed icon
- Monochrome ikona se přebarvuje podle tapety uživatele (Android 13+)
- **Konfigurace:** `ic_launcher_monochrome.xml`

#### 📱 Native UX
- Splash screen API
- Edge-to-edge layout (status bar transparent)
- Predictive back gesture (Android 14+)
- Native sharing (smlouva → WhatsApp, e-mail)
- **Pluginy:** `@capacitor/splash-screen`, `@capacitor/share`, `@capacitor/status-bar`

## 4. Technická architektura

```
DocFlow.aab (Capacitor)
├── android/                       ← native shell (Capacitor generuje)
│   ├── app/src/main/
│   │   ├── AndroidManifest.xml   ← oprávnění
│   │   ├── res/                  ← ikony, barvy, themes
│   │   └── assets/public/        ← bundleovaný web
│   └── build.gradle              ← targetSdk 35
│
├── public/                        ← váš web (bundlovaný)
│   ├── index.html                ← upravený pro Capacitor
│   ├── qrcode.min.js
│   ├── sw.js
│   └── icon-*.png
│
├── capacitor.config.ts           ← konfigurace
└── package.json                  ← závislosti

Backend (zůstává na Firebase):
├── 🔐 Firebase Auth              ← login
├── 💾 Firestore                  ← data
├── ⚡ Cloud Functions            ← familyInvite
└── 🔒 Secret Manager             ← Resend klíč

Web hosting:
└── 🌐 banbosh-smlouvy.web.app    ← landing page (po launch)
```

## 9. Native vs. Web detekce

V `index.html` přidáme:

```javascript
import { Capacitor } from '@capacitor/core';

const IS_NATIVE = Capacitor.isNativePlatform();

if (IS_NATIVE) {
  // Použij native pluginy: BiometricAuth, Camera, Push, atd.
} else {
  // Použij Web API: WebAuthn, getUserMedia, Web Push
}
```

Tím **stejný kód** funguje v Capacitor apce i v případném webu (kdybyste se rozhodl ho ponechat).

## 11. Rizika a mitigace

| Riziko | Pravděpodobnost | Mitigace |
|--------|-----------------|----------|
| Capacitor build padá | Střední | Časté commity, testování po každé změně |
| Plugin nekompatibilita | Nízká | Použijeme jen ověřené pluginy |
| Google Play zamítne | Nízká | Strict review checklist před uploadem |
| Bugy v native funkcích | Vysoká | 7+ dní closed testing |
| Testeři ztratí zájem | Střední | Komunikace s TestersCommunity |
| Časový skluz | Vysoká | Buffer 1 týden v plánu |
